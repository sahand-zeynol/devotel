<p align="center">Data transformer with Nestjs & postgres.</p>

## Description

## Installation

```bash
$ npm install
```

## config the env

copy the `example.env` to `.env.development` , make sure to connect to database

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
